Calorie Calculator

Apa itu Calorie Calculator?

Calorie Calculator adalah alat yang digunakan untuk menghitung kebutuhan kalori harian berdasarkan aktivitas, umur, berat badan, tinggi badan, dan jenis kelamin. Alat ini membantu Anda menentukan kebutuhan energi harian untuk mencapai tujuan berat badan Anda.

Rumus Menghitung Kalori

Untuk menghitung kebutuhan kalori, Calorie Calculator biasanya menggunakan Rumus Harris-Benedict yang dimodifikasi:

Pria:

BMR = 88.362 + (13.397 x Berat(kg)) + (4.799 x Tinggi(cm)) - (5.677 x Usia)

Wanita:

BMR = 447.593 + (9.247 x Berat(kg)) + (3.098 x Tinggi(cm)) - (4.330 x Usia)

Hasil BMR kemudian dikalikan dengan faktor aktivitas untuk mendapatkan Total Daily Energy Expenditure (TDEE):

  • Sedentari (tidak aktif): BMR x 1.2
  • Aktivitas ringan: BMR x 1.375
  • Aktivitas sedang: BMR x 1.55
  • Aktivitas berat: BMR x 1.725
  • Aktivitas sangat berat: BMR x 1.9

Cara Menggunakan Hasil Perhitungan

Gunakan nilai TDEE untuk merencanakan pola makan dan mempertahankan energi kalori sesuai kebutuhan Anda. Jika Anda ingin menurunkan berat badan, konsumsi kalori di bawah nilai TDEE. Jika ingin menaikkan berat badan, konsumsi kalori di atas nilai TDEE. Selalu konsultasikan dengan ahli gizi atau dokter untuk rencana diet yang tepat.
